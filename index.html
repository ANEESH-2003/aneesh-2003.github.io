Determining Gender By Name
Aneesh Khunteta, Dhruv Sharma, Kumar Pratyush,Atul Ratnawat(2022)
Advisor: Prof Sakthi Balan

description | report | Dataset

Large Internet companies no longer require gender information during account creation. However, predicting demographic data like gender and age remains a challenge, leading to unintentional gender and age bias in recommender systems. To address this, researchers investigated the difficulty of determining a user's gender based on their given name. After analyzing thousands of user names, they found that genders can be classified successfully using name compositions. Character-based machine learning models were developed, which significantly outperformed baseline methods in accurately determining user gender. Additionally, incorporating last names with first names further improved the classification performance.
